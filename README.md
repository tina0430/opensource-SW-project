* 초창기에 두서없이 만든 프로젝트이기 때문에 코드정리는 안되어있고 엉성할 수 있으니 참고용으로만 사용해주세요.
* 본 프로젝트는 이클립스를 기반으로 합니다.

## Tutorial
포함된 사항
```
DB(SQLite)
Home Widget
Paint
PasswordManager
RSS
Preferences
AlarmManager
```

아래와 같은 기능을 제공합니다
```
- 그리기 메모( 손글씨 , 낙서 )
- 메모위젯
- 메모작성
- 메모공유
- 메모검색
- 가져오기/내보내기
- 어플리케이션 잠금기능(암호기능)
- 어플꾸미기 기능
- 다양한 편의 옵션
```
